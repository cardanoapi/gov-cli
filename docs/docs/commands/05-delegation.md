# Delegation

Delegates votes to a specified DRep.

```
Usage: gov-cli delegate [delegation-target]
```


Valid options for delegation-target are :
 - `no-confidence`
 - `abstain`
 - `self` : delegate to your own drepId. 
 - `drep_id` : bech32 or hex encoded value  of drep_vkey_hash
